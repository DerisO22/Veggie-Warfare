# Computer Networking Final Project

### Tech Stack
- React Three Fiber
- TypeScript
- Express and Node
- Socket.io
- Google Firebase (Maybe)

## Project Set Up
1. Clone repository
2. Open Project in VS Code
3. Open multiple terminals in VS Code. One for frontend, server, and git stuff.

### Frontend
1. cd to frontend and run `npm install`
2. Then run `npm run dev`
3. Open localhost in browser

### Backend
1. cd to backend and run `npm install`
2. Then run `node server.js`
